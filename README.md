# University Scheduling Knowledge Base

## Prolog Setup (VS Code)

1. Install SWI-Prolog:  
   https://www.swi-prolog.org/

2. Install VS Code extension:
   - `swi-lsp` (recommended)

3. Verify installation:

```bash
swipl
```

Run a Prolog file:

```bash
swipl main.pl
```

Reload code inside Prolog:

```prolog
?- [main].
```
